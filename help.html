What is Html? What is Css

HTML is not a programming language, it is called Hyper Text Mark Up Language. Mark Up Language is a set of Mark Up tags. How different parts of a web page are displayed in a browser is expressed in HTML using Mark Up tags.
CSS stands for Cascading Style Sheets CSS describes how HTML elements are to be displayed on screen, paper, or in other media
